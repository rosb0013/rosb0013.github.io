What is Bootstrap?

Bootstrap is free. Boostrap is open source. Bootstrap harnesses the power of CSS. Bootstrap is a frontend framework to aid in quick development and deployment of websites. Bootstrap is also responsive, which means developing content that adjusts to multiple screen sizes, is a built-in and easily controlled by the developer for output perfection.

Getting Started!

Getting started with Bootstrap is as easy as copy and paste. Head on over to GetBootstrap.com, scroll down to the STARTER TEMPLATE and then copy/paste the provided code directly into your blank HTML doc. Voila! You now have access to Bootstrap and all its included customizations and components!

The Bootstrap GRID utilizes a 12 column, Flexbox grid system to provide developers with a powerful, felxible layout system. The GRID supports 6 responsive breakpoints for various screen sizes. Breakpoints are media queries that affect the container at pre-defined sizes.
